# Memory Bank Archive

このフォルダには、Memory Bankファイルの履歴アーカイブが保存されています。

## 目的

Memory Bankファイル（`activeContext.md`、`progress.md`等）は、プロジェクト進行に伴って大量の履歴が蓄積され、ファイルサイズが肥大化します。

このアーカイブフォルダは、以下の目的で使用されます:

1. **完全な履歴の保存**: 過去のすべての完了タスク、実装詳細、意思決定の記録を保持
2. **現行ファイルのシンプル化**: 現在の状態と次のタスクのみを現行ファイルに残す
3. **読み込み速度の向上**: ファイルサイズを削減し、Claude Codeの読み込み制限（25,000トークン）以内に収める

## アーカイブ構造

各アーカイブは期間ごとにサブディレクトリに整理されます:

```
archive/
├── README.md (このファイル)
├── 2025-10-28_to_2025-11-03/
│   ├── README.md (期間の説明)
│   ├── activeContext_archive_2025-10-28_to_2025-11-03.md
│   └── progress_archive_2025-10-28_to_2025-11-03.md
└── (将来のアーカイブ)/
```

## 利用方法

過去の詳細な履歴が必要な場合は、該当期間のアーカイブファイルを参照してください。

各アーカイブサブディレクトリのREADME.mdに、その期間の概要が記載されています。

## 運用方針

- **現行ファイル**: 最新状態のみ（現在のPhase、次のタスク、重要な教訓）
- **アーカイブ**: 上記以外のすべての履歴情報

定期的に履歴が蓄積したら、同じパターンでアーカイブ化を実施します。
